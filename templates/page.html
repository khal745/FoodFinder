FoodFinder

Transfer learning powered!

Ever had trouble finding out if there's food in an image?

Well worry no more! Using the power of a Transfer Learning model, you can find out if there's food in your picture!

Trained using a ResNet50 base, this Transfer Learning model achieves approximately 95% accuracy in determing whether an image contains food or not. Developed as a part of the NZMSA programme, if you'd like to learn more about the model used, please click here to see a Azure notebook detailing the creation of the model.
